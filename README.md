# ⛽ 燃費トラッカー

ガソリンスタンドで **3秒で入力が完了する** 自分専用の燃費管理PWAアプリ。

**→ [アプリを開く](https://hajihajitsubo.github.io/fuel-tracker/)**

## 機能

- **給油データ入力** — 給油日・総走行距離(km)・給油量(L)・合計金額(円)
- **自動計算** — 実燃費(km/L)・トリップ走行距離・ガソリン単価(円/L)をリアルタイムで表示
- **給油履歴** — 新しい順に一覧表示。各記録の削除も可能
- **オフライン対応** — Service Workerによりネット接続なしで動作
- **データ永続化** — LocalStorage保存のためサーバー不要・維持費0円

## 使い方

### iPhoneにインストール

1. SafariでアプリのURLを開く
2. 画面下の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」→「追加」

### 初回使用

最初の給油記録は**基準点**として保存されます。次回給油時から実燃費・走行距離が自動計算されます。
